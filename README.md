# GroceryHandler

# Author
  Victor, Datastax intern

IOS app for facilitating the accouting of shared groceries.

This app is intentended to be used for when multiple people do groceries together,
have one person pay for the whole thing, and have the other people reimburse the payer.

Goal: All user has to do is take picture of the grocery ticket
and specify who uses which item.
The app will then inform the user how much each person owes to the person who paid.
